## What is a memory?
- **Random Access Memory (RAM) is the most common type**
	- Not the only kind of memory
- **RAM is not referring to hard drive or SSD storage**
	- Don't mix the two terms
	- Data is stored permanently on the drive
- **Data and programs can only be used when moved to RAM**
	- RAM is a temporary, high-speed storage that we use when we are executing applications and performing calculations on our computer.
	- Users can only work with data this way if it has been loaded into the RAM of the computer.

### What is RAM? [^11]
- **Random Access Memory**
	- Computer's short-term, high-speed memory
	- Temporarily stores data that CPU needs right now to perform active tasks
- **Function**
	- Programs like web browsers, games, or Word docs are loaded into RAM so they can run quickly & smoothly
	- The more RAM you have, the more tasks your computer can handle at once
- **Volatile Memory**
	- Data in RAM is temporary
	- It's lost when the power is turned off
- **Comparison to Storage Devices**
	- RAM = short-term, fast, temporary
	- Hard Drives/ SSDs = long-term, slower, permanent storage

## RAM Form Factors
- <mark>Form Factor = Physical Design</mark>
### DIMM (Dual In-Line Memory Module)
![537](<./assets/3.3 An Overview of Memory/file-20260310152809239.png>)[^2]
- **A single memory module**
	- Instead of memory sticks to install or remove from the motherboard
- **DIMM**
	- <mark>Dual Inline Memory Module</mark>
	- Electrical contacts are different on each side
- **64-bit data width**
- Can be installed at one time by pushing entirely into a single slot.
	- To remove, pull back the locks on the side of the module, then simply pull out the module out of the motherboard.
- **Pin Count:**
	- DDR3: 240 pins
	- DDR4 / DDR5: 288 pins

### SO-DIMM (Small Outline DIMM)
![](<./assets/3.3 An Overview of Memory/file-20260310154349129.png>)[^3]
- <mark>Small Outline Dual In-Line Memory Module</mark>
	- About half the width of a DIMM
	- Compact RAM
- **Used in laptops and mobile devices**
	- Varies by type
- Similar to a full-size DIMM, but not interchangeable due to physical differences. [^10]
- **Pin Count:**
	- **DDR3**: 204 pins
	- **DDR4/ DDR5**: 260 pins

### Form Factor Comparison
![](<./assets/3.3 An Overview of Memory/file-20260310154514143.png>)[^4]
- **DIMM**
	- Ideal for systems with large cases & motherboards supporting full-size memory slots
- **SODIMM**
	- Optimized for tight spaces & portable systems where a full-sized DIMM would not fit

## Dynamic Random Access Memory
- **DRAM**
- **The memory chips on the DIMM**
- **Dynamic**
	- Needs constant refreshing
	- Without refreshing, the data in memory disappears
- **Random access**
	- Any storage location can be accessed directly
	- Unlike magnetic tape

## SDRAM
- **Synchronous DRAM (SDRAM)**
- **SDRAM is synchronous with the common system clock**
	- Queue up one process while waiting for another
		- It helps keep track of the standard rate of data that is being transferred back and forth
	- Classic DRAM didn't wait for a clock signal

## Comparison table between DRAM and SDRAM

| **Feature**       | **Legacy DRAM**                        | **SDRAM**                                 |
| ----------------- | -------------------------------------- | ----------------------------------------- |
| **Clock Sync**    | Asynchronous (No clock)                | Synchronous (Matches System Clock)        |
| **Performance**   | High Latency (Wait states)             | Low Latency (Pipelining)                  |
| **Analogy**       | A person walking at their own pace.    | A soldier marching to a drumbeat.         |
| **A+ Exam Focus** | Understanding why "Refresh" is needed. | Understanding it's the basis for all DDR. |

## SDR vs DDR
![](<./assets/3.3 An Overview of Memory/file-20260310160831556.png>)[^5]
- <mark>SDR === Single Data Rate</mark>
	- Older style of memory to transfer information from memory into CPU and then back again.
	- Uses clock rate to manage the transfer of data.
		- 1 single clock cycle results 1 bit of data being transferred.
- <mark>DDR === Double Data Rate</mark>
	- Can transfer information at the top and bottom of the clock cycle
		- Effectively *doubling* how much information can be transferred from memory into CPU and vice versa, the same amount of time.
	- 1 clock cycle = 1 data transfer

## DDR Generations
- DDR (Double Data Rate)
	- Transfers two data bits per clock cycle, doubling speed compared to single data rate (SDR)
	- **Core Concept**: <mark>More data per second = better performance</mark>
- The number is a designated version of the DDR that's installed.

### Progression of DDR Technology
#### Double Data Rate 3 (DDR3) SDRAM
![](<./assets/3.3 An Overview of Memory/file-20260310162210864.png>)[^6]
- Increased bandwidth; still found in older PCs
- **Twice the data rate of DDR2**
	- Larger chip capacities
- **[No backwards compatibility](<###Compatibility Note>)**
	- Speed brings sacrifice
- If your motherboard supports DDR3 memory, your old DDR2 memory will not be compatible.

#### Double Data Rate 4 (DDR4) SDRAM
![](<./assets/3.3 An Overview of Memory/file-20260310162431581.png>)[^7]
- Mainstream in most modern systems; faster and more efficient
- **Speed increases over DDR3 technologies**
	- Faster frequencies
- **[No backwards compatibility](<###Compatibility Note>)**

#### Double Data Rate 5 (DDR5) SDRAM
![](<./assets/3.3 An Overview of Memory/file-20260310164122283.png>)[^8]
- Latest generation; supports much higher speeds & lower voltage
	- Designed for high-performance computing
- **Faster data transfers between the memory module and the CPU on the motherboard**
	- Higher speeds than DDR4
- **The key has moved**
	- [No backwards compatibility](<###Compatibility Note>)

### Compatibility Note
- <mark>Not interchangeable</mark>: Each DDR type has a unique pin layout & voltage
	- DDR4 can't fit in a DDR3 slot

### Performance Trends
- Each generation offers:
	- High transfer rates (measured in MT/s - Mega Transfers per second)
	- Lower voltage consumption for better efficiency
	- Improved reliability & bandwidth handling

## DDR and SO-DIMM Comparisons
![413](<./assets/3.3 An Overview of Memory/file-20260310161817937.png>)[^9]
- There are small keys or holes at the bottom of the memory module, to correlate with the key on their respective motherboards
- This prevents the memory module from being installed on the wrong motherboard.
