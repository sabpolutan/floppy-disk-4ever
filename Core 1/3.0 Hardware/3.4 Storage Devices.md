## Storage Devices
![487](<./assets/3.4 Storage Devices/file-20260311183756339.png>)[^2]
### What is a Storage Device?
- A piece of hardware used to hold data, temporarily or permanently
- **Two main categories**
	- **Primary storage (e.g., RAM)**
		- Temporary data storage is used while the computer is running
		- ***Volatile***: Data is lost when power is turned off
	- **Secondary Storage (e.g., HDDs, SSDs)**
		- Permanent data storage that retains info even when powered off
- **Need some way to store information when the computer is off**
	- RAM requires power to maintain the data
- Many different ways to store data
	- Hard disk drives (HDD)
	- Solid-state drives (SSD)
	- Flash drives
	- Memory cards
	- Optical drives


## Hard disk drives (HDDs)
- **Non-volatile magnetic storage**
	- Rapidly rotating platters
- **Random-access**
	- Retrieve data from any part of the drive at any time
- **Moving parts**
	- Spinning metal platters, moving the actuator arm
	- A read / write head
	- Mechanical components limit the access speed
	- Mechanical components can also break at any time
	- The top of the drive should remain dust-free
- **Data Storing Method**: Info is written to & read from magnetic platters as they spin

### Spindle speeds
- <mark>RPM = Revolutions Per Minute</mark>
![](<./assets/3.4 Storage Devices/Screenshot 2026-03-11 at 6.29.32 PM.png>)

| Rotational Speed (rpm) | Average Rotational Latency |                                                                                         |
| ---------------------- | -------------------------- | --------------------------------------------------------------------------------------- |
| 15,000                 | 2 ms                       |                                                                                         |
| 10,000                 | 3 ms                       | High-performance drives used in servers or enterprise systems; largely replaced by SSDs |
| 7,200                  | 4.16 ms                    | Faster performance; standard in most desktop PCs                                        |
| 5,400                  | 5.55 ms                    | Slower, quieter, energy-efficient; common in laptops or backup drives                   |

[^1]
- The faster the spindle rotates, the lower the latency will be
	- Resulting in faster read/write information

## Drive size comparison (form factors)
![495](<./assets/3.4 Storage Devices/Screenshot 2026-03-11 at 6.39.19 PM.png>)[^3]
- Desktop computers don't need to worry about the size of the storage device as there is a lot of room to store it
- Smaller storage device will help save a lot of space especially for laptops and mobile devices
- Modern computers may already be using the 22mm SSD

### HDD Form factors (Physical size)
- **3.5-inch**: Standard in desktop PCs & servers – larger & higher capacity
- **2.4–inch**: Common in laptops & portable external drives – smaller & more power-efficient
- **Adaptability**: 2.5-in drives can be used in desktops with an adapter bracket


## Solid-state drives (SSD)
- **Non-volatile memory**
	- No moving parts
- **Very fast performance**
	- No spinning drive delays


### 2.5" SATA interface
![444](<./assets/3.4 Storage Devices/file-20260311184417619.png>)[^4]
- No moving components inside
- They simply consist of [memory modules](<./3.3 An Overview of Memory.md>), where the information is being stored

## PCIe storage interfaces
- <mark>PCIe = Peripheral Component Interconnect Express</mark>
- Need some way to increase the throughput of the SSDs, since *a traditional SATA drive has reached a maximum limit of connection (6 gigabits per second)*
- **High speed for SSD communication**
	- Used before M.2
- **Connect directly to the motherboard**
	- Provides power
	- High throughput
- <mark>PCIe = PCI express</mark>
	- Modern PCIe standards can transfer data at 64 gigabits per second per lane
- However, not every device has a PCIe bus to connect to
	- Laptops, for example, don't have the room to be able to install a full-size adapter card

## NVMe
- **SATA was designed for hard drives**
	- Uses ***ACHI (Advanced Host Controller Interface)*** to move drive data to RAM
	- SATA revision 3 throughput up to 6 Gbps
	- SSDs need a faster communication method
- <mark>NVMe (Non-volatile Memory Express)</mark>
	- Designed for SSD speeds
	- Lower latency, supports higher throughputs
	- Uses the PCI Express bus
	- NVMe using M.2 has a theoretical transfer speed of 20 Gbps
	- Offers the fastest transfer speeds for SSDs

## Serial Attached SCSI (or SAS)
- A serialized version of the SCSI technology
- Faster and more reliable than SATA
- Found in enterprise servers and workstations
- **The latest generation of SCSI technology**
	- Increased storage throughput
	- Serial communication
	- SCSI protocol
	- <mark>SAS - Serial Attached SCSI</mark>
- **The control and management of SCSI**
	- Useful in large storage arrays
- **An upgrade to SATA throughput**
	- SATA can communicate at 6 gigabits per second
	- High-end SAS can reach 22.5 gigabits per second
- **Good for large storage arrays of spinning hard drives**
	- You may want to focus on using SAS arrays with SAS drives to get the highest throughput possible for that hard drive technology


## mSATA (Mini-SATA)
![](<./assets/3.4 Storage Devices/file-20260311190913867.png>)[^5]
- **Shrink the SATA drive into smaller devices**
	- Same data, different form factor
	- Great for laptops and mobile devices
- **Smaller than 2.5" SATA drives**
	- No spinning drives
	- Allows for different form factors
- **Was used briefly**
	- Quickly replaced by the *m.2* standard
- mSATA solved some problems, especially the limited size on a system board and the need for a higher-speed bus.

## M.2 interface
![406](<./assets/3.4 Storage Devices/file-20260311192738012.png>)[^6]
- **Smaller form factor**
	- No SATA data or power cables
- **Can use a PCI Express bus connection**
	- 4 GB/second throughput or faster when using NVMe PCIe x4
- **Different connector types**
	- Needs to be compatible with the slot key/ spacer
	- B key, M key, or B and M key
	- Some M.2 drives will support both
![](<./assets/3.4 Storage Devices/file-20260311194142908.png>)[^8]
- Can plug directly into the motherboard

### B-key and M-key
![](<./assets/3.4 Storage Devices/file-20260311193836198.png>)[^7]
- **M.2 drive doesn't guarantee NVMe interface**
	- Your M.2 interface may be using AHCI
	- Check your documentation
- **Your motherboard may only support one type of M.2 key**
	- Check documentation


## Flash drives

- **Flash memory**
	- ***EEPROM (Electrically erasable programmable read-only memory)***
	- Non-volatile memory
	- No power required to retain data
- **Limited number of writes**
	- Can still read the data
- **Not designed for archival storage**
	- Easy to lose or damage
	- Always have a backup

## Optical drives

- **Small bumps read with a laser beam**
	- Microscopic binary storage
- **Relatively slow**
	- Archival media
- **Many different formats**
	- CD-ROM, DVD-ROM, Blu-ray
- **Internal and external drives**
	- For those uncommon application installations
